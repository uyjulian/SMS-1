3.6. Correndo ficheiros de video a partir de um dispositivo USB ligado à PS2.

Pode utilizar um dispositivo USB ligado à frente da sua PS2 para ver videos ou tocar os seus MP3.

#SMS reads files from a USB drive using a USB driver built-in to the SMS application. There's no need to install any additional files to enable USB support in new versions of SMS (V1.7 or later). Older versions required a 3rd party USBD.IRX file.#

#When you plug a compatible USB drive into the front USB port on the PS2, you will see a new icon as shown below.# Pressione DIREITA para seleccionar este icon e navegar no dispositivo USB.

Figura 3.31. SMS com USB activado.


Como habitualmente, pressione X para correr o ficheiro seleccionado.
