3.1.4. Menu do Leitor

O menu do leitor (SMS v1.5 para cima), deixa que você altere as opções a meio da vizualização. Pressione START durante a vizualização para aceder ao menu do leitor.

Figura 3.2. Screenshot mostrando o menu do leitor.


Item do Menu

Descrição

Linguagem:

X para mudar a linguagem de audio (para ficheiros avi com múltiplas pistas de áudio).

Ecrã:

X para rodar entre os 5 modos de vizualização disponíveis: 'letterbox', 'pan-scan 1/2/3', e ecrã completo.

O modo 'letterbox' mostra o video com o seu comprimento original com bandas pretas no topo e fundo do ecrã. 'Pan-Scan' amplia o video, utilizando mais linhas verticais do ecrã, mostrando uma imagem maior no centro. O aspecto negativo é que os lados esq. e dir. da imagem são cortados. Quanto maior o nivel de 'pan-scan', maior a ampliação e o corte. O modo de ecrã completo estica a imagem para que esta caiba o ecrã da TV - isto é útil ao vizualizar um video em formato 'widescreen' numa TV 'widescreen'.

Desligar automaticamente: (temporizador)

Desliga a PS2 automaticamente depois de um tempo especificado. O tempo começa a contar a partir do último butão pressionado.

X para aumentar

O para diminuir

auto = desligar quando o filme acabar.

3.1.5. Barra de Busca

Novo na versão 1.5 é a Barra de Busca, acessível durante a vizualização pressionando SELECT. Quando activada, a barra de busca mostra a posição corrente do video. Isto pode ser ajustado pressionando ESQUERDA/DIREITA, depois a vizualização pode ser resumida desse ponto pressionando X ou START.

Existem várias maneiras de vizualizar ficheiros de video:

  • Através da Rede Local

  • A partir de um CD

  • A partir de um DVD-R

  • A partir do Disco Rigido da PS2

  • A partir de um dispositivo USB ligado à PS2 (também chamado dispositivo de armazenamento USB).
