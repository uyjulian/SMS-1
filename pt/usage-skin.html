3.9. "Skinning"

O SMS permite uma alteração básica do 'skin' do programa. Isto significa que pode alterar a imagem de fundo do navegador para uma imagem à sua escolha.

Nota

Actualmente só é possível alterar a imagem de fundo do navegador, não é possível alterar as posições dos controles e icons no ecrã.

3.9.1. Criar uma Skin

O ficheiro para skin é uma imagem codificada em DivX5xx/XviD, com o nome de mc0:/SMS/SMS.skn. Um pequeno programa de Windows (pic2sms.exe) foi criado para facilitar a criação destes ficheiros a partir de simples ficheiros de imagem. Primeiro necessitará de um ficheiro de imagem, neste exemplo foi utilizado uma fotografia das minhas férias (jpeg).

É necessário criar um ficheiro bitmap, com dimensões < 800x600. Se precisar de ajuda a perceber onde se localizão os elementos do sistema em relação a estas dimensões, pode fazer o download de um ficheiro de 'template' aqui: http://www.ps2-scene.org/forums/attachment.php?attachmentid=8751&d=1131850982. [Atenção - As alterações entre o SMS 1.5 e o 1.6 podem não estar reflectidas neste 'template'.] Este ficheiro está em formato de Photoshop (.psd), mas se não tiver o Photoshop pode abri-lo utilizando o GIMP (editor de imagens gratuito) http://www.gimp.org/. Quando tiver pronto, grave o ficheiro como bitmap de camada simples (single layer bitmap) (.bmp) num directório vazio no seu PC.

De seguida, faça o download do gerador de skin pic2sms.exe de http://home.casema.nl/eugene_plotnikov/, e extraia o ficheiro pic2sms.exe do ficheiro pic2sms.rar utilizando o WinRAR. Copie-o para o mesmo directório do seu PC. Depois faça duplo clique no ficheiro. Se este abrir à primeira vez sem erros está tudo bem. Se aparecer uma mensagem a dizer 'cannot find xvidcore.dll', feche o programa, e faça o download do xvidcore.dll de http://www.dll-files.com/dllindex/dll-files.shtml?xvidcore ou de qualquer outro local (procure no google). Copie o ficheiro xvidcore.dll para o mesmo directório do pic2sms.exe. Quando executar o programa novamente não deverão ocorrer nenhuns erros.

Agora clique em 'open' e escolha o ficheiro bitmap que criou.

Deverá ver a sua imagem aparecer na janela. Clique no butão de de salvar para criar um ficheiro com extensão .skn no mesmo directório. E pronto, a sua skin está criada!

3.9.2. Aplicando a Skin

Para aplicar a skin (quer tenha sido feita por si ou tenha feito o download da internet), precisa do seguinte:

Seleccionar o ficheiro SMS.skn. Se o seu ficheiro tem o nome xxxxxx.skn, mude-lhe o nome para SMS.skn.

Agora necessita de transferir o ficheiro para mc0:SMS/SMS.skn (i.e para o cartão de memória na ranhura 1 da sua PS2). Assumimos que esteja a transferir por FTP. Na PS2, Inicie o uLaunchELF (ou outro programa que permita correr ELF's), e seleccione o programa de FTP (e.g. PS2NET).

Assim que o servidor de FTP estiver a correr na PS2, vá ao seu PC, inicie o seu cliente de FTP (e.g. FileZilla), e estabeleça ligação com a PS2.

Arraste o ficheiro SMS.skn no directório mc0:/SMS/SMS.skn. Quando estiver completo, desligue o cliente de FTP e reinicie a PS2.

Figura 3.43. Transferindo ficheiro de skin para a PS2.


Na sua PS2, inicie o SMS normalmente, depois pressione START para ver o menu de configuração, depois seleccione 'browser settings (Opções do Navegador)'.

Figura 3.44. Menu Browser settings (Opções do Navegador).


Seleccione a opção 'Use background image' (usar imagem de fundo) para 'ligar' a skin. Deverá ver o fundo mudar automaticamente.

Figura 3.45. Menu Browser settings (Opções do Navegador) com skin activada.


Se quiser mudar a skin, basta copiar um novo ficheiro SMS.skn para substituir o outro.

Figura 3.46. Skin de Exemplo


Nota

O SMS 1.5 e anteriores versões têm um formato de skin diferente que utilizava um ficheiro mc0:/SMS/skin.sms em vez de mc0:/SMS/SMS.skn. Existia também uma versão diferente do pic2sms.exe (gerador de skins). O antigo gerador de skins já não se encontra disponível para download. Se ainda tiver uma versão antiga do SMS e do pic2sms.exe, aconselhamos que faça um upgrade para as versões mais recentes. Ambos podem ser encontrados no site do SMS.
