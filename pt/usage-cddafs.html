3.3. Correndo ficheiros de video a partir de um CD-R (without a modchip)

Nota

Para correr ficheiros de video a partir de um CD, precisa de gravar o CD num formato especial. A PS2 não consegue ler data de CD's com formato ISO. O “formato especial” é criado utilizando o CDXplorer.exe.

3.3.1. Instruções curtas

  • Faça o download do CDXplorer.exe do SVN;

  • Execute-o no Windows;

  • Arraste os ficheiros que deseja do Explorador do Widnows para a aplicação (ou use o butão '+' na barra de ferramentas);

  • Salve a compilação - ficará com um ficheiro cue e um bin;

  • Feche a aplicação;

  • Execute uma aplicação de gravação de CD's (i.e. Nero);

  • Abra o ficheiro .cue;

  • Se estiver a usar o Nero 5 seleccione a opção Disk-at-once/96;

  • Grave o CD;

Boa Sorte

Eugene

3.3.2. Instruções detalhadas

A PS2 não modificada não consegue ler data de um CD-R normal. Isto deve- se a um mecanismo de protecção anti-cópia interno. Para dar a volta a isto, o criador do SMS desenvolveu um pequeno truque que leva a PS2 a pensar que o CD-R é um CD de áudio. A PS2 consegue ler o CD-R se este for um CD de áudio. O principio básico consiste em criar um disco no formato CDDAFS, que é essencialmente um disco que se parece suficientemente com um CD de áudio para que a PS2 o aceite. O disco contém o seu ficheiro .avi escondido dentro de pistas de áudio, e o SMS converte-as à medida que as lê. Esperto huh?

O criador programou um utilitário (CDXplorer.exe) para criar os ficheiros .cue/.bin do ficheiro .avi, pronto para ser gravado em CD-R. Depois, desde que o software de gravação de CD's consiga ler ficheiros .cue/.bin, pode gravar esses ficheiros num CD-R.

Importante

Não basta copiar os ficheiros .cue/.bin para um CD em formato ISO. Isto não funciona. Tem de usar um software de gravação de CD que funcione com ficheiros .cue/.bin. Os únicos conhecidos de momento são o Nero e o cdrdao.

Faça o download do CDXplorer.exe a partir de http://ps2dev.org/ps2/Projects/Softwares (CDXplorer.exe pode ser encontrado dentro do ficheiro SMS.zip nesta página).

Execute a aplicação fasendo duplo clique sobre ela (não é necessário instalar).

Figura 3.20. Screenshot do CDXplorer.


Assim que o CDXplorer esteja a correr, arraste 1 ou mais ficheiros AVI para a janela.

Figura 3.21. Screenshot do CDXplorer mostrando ficheiro AVI testvid.avi depois de este ser arrastado do Explorador do Windows.


Clique file->SaveAs, insira um nome (e.g. 'test') e pressione <enter>. Isto salvará dois ficheiros: test.cue e test.bin no directório actual.

Pode agora fechar o CDXplorer.

De seguida execute o seu programa de gravação de CD's. Tem de ser um software que permita carregar ficheiros .cue. Carregue o ficheiro .cue.

3.3.2.1. Nero 5

Se estiver a usar o Nero 5, seleccione 'Disk-at-once/96', e grave o CD-R

3.3.2.2. Se não tiver o Nero 5

Se você não tiver o Nero 5 (eu não tenho), pode utilizar outro software (grátis) para gravar CD's em modo disc-at-once. O único porgrama que eu consegui usar com sucesso foi o cdrdao (CDR Disc At Once), que é uma aplicação de linha de comando para gravar CD's. Está disponivel para Linux e para Windows. Se encontrar outras aplicações que sejam mais fáceis de utilizar, por favor refira os detalhes no forum do SMS. Instalação do cdrdao pode ser feita da seguinte maneira:

Linux (Fedora Core 4)

# yum install cdrdao

Windows

Downloade e instale http://ovh.dl.sourceforge.net/sourceforge/cdrdao/cdrdao-1.1.5.bin.x86.win32.zip.

Para gravar o CD execute o seguinte comando: (Note a opção --force que é necessária para efectuar a gravação sem ligar ao aviso 'mais curto que 4 segundos')

cdrdao write --speed MAX --force test.cue

Deverá aparecer algo como isto:

E pronto. Deverá ter um CD com um ficheiro AVI compativel com o SMS. Este Cd não funcionará no seu PC. Aliás, se o colocar no PC, ele provavelmente tentará tocá-lo como se fosse um CD de áudio e apenas ouvirá 'ruido'. Inicie o SMS, depois coloque o CD na PS2 e deverá ver o seguinte:

Pode agora navegar até ao ficheiro que copiou, e pressionar X para iniciar a leitura da maneira habitual.

Se não vir o ficheiro, e a sua PS2 fizer um barulho constante, ou o seguinte icon 'vermelho', então existe um problema com o disco. Provavelmente você tem um CD-R queimado (estragado), e terá de tentar novamente. Repita as instruções mas com uma velocidade de gravação mais baixa. Se utilizou um CD-RW, tente agora com um CD-R.
